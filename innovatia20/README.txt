INSTALL GUIDE

For the the installation of the prototype Inovatia server, follow these steps:
1. Copy all Innovatia files into the HTTP directory of your PHP server.
2. Run the innovation20.sql scrtipt on a MySQL DB.
3. Update connector.php with the details of that DB.

After that you should be able to use Innovatia server with Innovatai mobile.
